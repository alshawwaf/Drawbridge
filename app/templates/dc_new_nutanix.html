Define the inventory the mock Nutanix Prism (REST) will serve. Point a SmartConsole Nutanix Data Center object at the portal to import VMs (with their IPs) and the Categories (Prism key/value tags) applied to them. Serves both Prism v3 and v4 (CloudGuard probes v4, falls back to v3).

Served at /api/nutanix/v3/vms/list (and the v4 /api/vmm/v4.1/ahv/config/vms); the IP comes from the VM's NIC. The pipe part is the VM's Categories — Prism's key=value tags. The Category tree (Key → Value → VMs) is built from these, so a category resolves to the VMs carrying it.
