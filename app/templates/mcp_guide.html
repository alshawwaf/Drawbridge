Safety

Every call needs Authorization: Bearer <key>, where the bearer is an active MCP-scope API key (generated above, or in Settings → API keys — individually revocable, SHA-256-hashed at rest). Create or revoke takes effect with no redeploy — no active key → 503 (endpoint disabled), wrong/missing key → 401. decide_access, the analyse tools, and the correlate/coverage tools are read-only. apply_access with publish=false dry-runs (validate then discard); publish=true is refused unless an admin enables Settings → MCP / agent → "Let the MCP agent publish to live policy". The engine's own guarantees still hold — an unknown/ambiguous service returns a review + suggestions, never a wrong call to the SMS.
